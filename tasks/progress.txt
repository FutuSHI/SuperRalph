## Codebase Patterns

- ralph.sh uses perl for placeholder replacement (replace_placeholder function). Use the same approach for any text substitution to maintain consistency and cross-platform compatibility.
- Discipline files are in disciplines/ directory. They are injected into CLAUDE.md.template via {PLACEHOLDER} syntax.
- Hook scripts must use CLAUDE_PLUGIN_ROOT or derive plugin directory for portable file references.
- All shell scripts should pass shellcheck validation.
- Use `${var##*/}` for portable path basename extraction instead of `echo | sed`.
- For hook scripts, use `${CLAUDE_PLUGIN_ROOT:-$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)}` to portably locate the plugin directory.
- Use `read_discipline()` helper pattern: try file first, fall back to inline string.
- Agent prompt templates live in agents/ directory. Referenced by CLAUDE.md.template.
- Sub-technique discipline files are referenced from the main discipline file (e.g., debugging.md links to root-cause-tracing.md).
